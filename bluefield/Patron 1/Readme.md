# Led Counter

A simple led "counter" in arduino. 

## Pattern:

Column: Led

Row: Phase

```
P1  : [0] [0] [0] [0] [0]
P2  : [1] [0] [0] [0] [0]
P3  : [1] [1] [0] [0] [0]
P4  : [1] [1] [1] [0] [0]
P5  : [1] [1] [1] [1] [0]
P6  : [1] [1] [1] [1] [1]
P7  : [1] [1] [1] [1] [0]
P8  : [1] [1] [1] [0] [0]
P9  : [1] [1] [0] [0] [0]
P10 : [1] [0] [0] [0] [0]
P11 : [0] [0] [0] [0] [0]
P12 : [1] [1] [1] [1] [1]
P13 : [0] [0] [0] [0] [0]
P14 : [1] [1] [1] [1] [1]
```
